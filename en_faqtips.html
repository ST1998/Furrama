ðŸ‘‰ Create folders on official Telegram ðŸ‘ˆ

( 1 ) If you have already downloaded Plus Messenger (based on Telegram) from Play Store (or from other sources), launch it and, from the Plus Messenger main screen, press the folder icon (top right), then press Add category option and insert the name Furrama, then press OK.

( 2 ) Then, select Furrama Pocket and Furrama groups. To facilitate everything, press the magnifying glass icon (top right), then write the word to search for (Furrama) and select what you have found about it. Repeat step 2 and after you are done, press the checkmark icon (top right). The confirmation notice will then appear, where it will show you everything you selected before and, if you are sure you have done everything right, press Yes.

( 3 ) Afterwards, you will be catapulted into the Categories screen, where you can select each category that you previously created. From that screen, you can add a new category by pressing the + icon (top right). This screen can be reached by holding down the folder icon from Plus Messenger main screen.

( 4 ) Optional: Next to the category name (in this case, Furrama), press the icon with three dots. Here are the following options: Edit category name, Edit category (you can choose which to exclude in that category), Delete (the category), Select chats (allows you to do everything it says in point 2) and Add to Home screen (I don't recommend it, since it doesn't work as it should).

( 5 ) Optional: From the Categories screen, press three dots icon (with no category name next to it and it is located at the top right). Here are the following options: Hide categories icon (not recommended), Export categories (allows you to create a small backup. I recommend saving it in a File Manager) and Import categories (you can restore the backup made previously. If you have saved it in the internal memory, select Browse your file system, otherwise, select Browse your external storage. Scroll and find the file indicated with the icon with cat written on it and you're done!).
